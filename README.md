## 四、火星强制设计硬约束
1、重力：0.38倍地球重力，结构、地基、机器人、混凝土全部重新适配
2、大气风沙：大气密度仅地球1%，沙尘带静电强磨蚀，所有设备必须全封闭防尘
3、宇宙辐射：无磁场无厚大气屏蔽，核心设施必须布置熔岩管内部岩层屏蔽
4、极端温差：地表-125℃至+60℃，熔岩管常年-20℃至0℃，普通橡胶塑料全部禁用
5、水体特性：水结冰体积膨胀9%，严禁自由水外露，全程闭环控温
6、地质风险：存在火星震、围岩蠕变、地下冰塌陷，核心建筑必须坐落完整玄武岩基岩
7、外援约束：地火窗口26个月一次，无现场运维无备件，关键系统必须多重冗余、支持本地3D打印自修
